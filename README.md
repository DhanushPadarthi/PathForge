## Getting Started

### Prerequisites
- Node.js (v18+)
- Python (v3.9+)
- MongoDB

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
